# Smart-Routine

Smart-Routine is the planning repository for RoutineOS, an offline-first Flutter routine, habit, and productivity app.

The app goal is to become a time-wise routine operating system where users can plan daily routines, track progress, recover missed habits, receive smart local reminders, and review productivity analytics without requiring a paid backend or paid AI API.

Start with:

- [plan.md](plan.md) for the implementation plan.
- [instruction.md](instruction.md) for agent working rules.
- [memory.md](memory.md) for persistent project progress notes.
